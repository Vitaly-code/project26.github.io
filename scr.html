Что такое JavaScript?

JavaScript - это сравнительно новый язык для написания сценариев, разработанный Netscape. При помощи языка JavaScript мы можем создавать интекрактивные сайты наиболее удобным и эфективным способом. В настоящем практическом пособии приведены примеры, которые по мнению автора, являются наиболее важными. Они смогут продемонстрировать возможности javascript и принципы организации языка. В этом пособии приводятся примеры того, что можно сделать, используя javascript а также как это сделать...

Как запустить скрипт, написанный на javascript?

Что требуется для того, чтобы запустить программу-скрипт. Все что для этого требуется - это обычный интернет-браузер, с активированной способностью распознавать javascript. Таким браузером, например может быть netscape navigator 2.0 и выше или internet explorer 3.0 и выше. Поскольку большинство пользователей сети интернет используют именно эти два браузера, то становится понятным, почему имено язык javascript является наиболее предпочтительным язком при решении задач улучшения качества web-страниц. Заметим также, что для понимания изложенного здесь материала требуется знание основ html - языка разметки, при помощи которого создаются web-страницы.
